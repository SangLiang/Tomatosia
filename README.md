# Tomatosia

---

### 1.介绍:
采用Electron开发,简单的番茄钟软件。web方面使用jqeury。

### 2.使用:
1. 基于node环境
2. 全局安装 electron 
    ``` npm install electron -g```

3. 全局安装 electron-forge
    ```npm install electron-forge -g```

4. 运行 ``` npm run start ```

5. 打包 ``` npm run make ```
